# React + TypeScript + Vite

### Tech
- React 18
- ViteJS - for dev server + bundle (with Rollup)
- Ant Design
- CSS Modules + LESS
- Redux + Redux Toolkit
- ESLint + Prettier
- chartjs - for Chart 

### Getting started

`yarn install` 
`yarn dev` - Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
